# HistInTerminal

A tool to show a histogram

## Install

   make
   make install

app will be installed in "~/bin".
You can edit "INSTALL_DIR" in Makefile to change the directory of installation.

## How To Use

Usage: hit [ values ]

[ values ] can be separated by white space, \t and \n.
